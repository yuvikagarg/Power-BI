# Power-BI
# 🛒 E-Commerce Sales Dashboard | Power BI

## 📌 Project Overview
This project is an interactive **E-Commerce Sales Dashboard** developed using **Power BI** to analyze sales performance, profitability, customer behavior, and payment preferences. The dashboard transforms raw sales data into meaningful business insights that help stakeholders make data-driven decisions.

---

## 🎯 Objective
The objective of this project is to:

- Monitor overall sales performance and profitability.
- Identify top-performing products and categories.
- Analyze customer purchasing behavior.
- Understand preferred payment methods.
- Compare profit trends over time.
- Support business decision-making through interactive visualizations.

---

## 📊 Dashboard KPIs

| KPI | Description |
|-----|-------------|
| Total Sales | Overall revenue generated |
| Total Profit | Overall profit earned |
| Total Quantity Sold | Number of products sold |
| Average Order Value (AOV) | Average revenue per order |

---

## 📈 Dashboard Insights

### 1. State-wise Sales Analysis
- Displays total sales generated by each state.
- Helps identify high-performing geographical regions.

### 2. Category-wise Quantity Analysis
- Compares quantity sold across product categories:
  - Clothing
  - Electronics
  - Furniture
- Helps understand customer demand.

### 3. Monthly Profit Analysis
- Shows profit trends by month.
- Helps identify profitable and low-performing periods.

### 4. Customer-wise Sales Analysis
- Displays contribution of individual customers.
- Helps identify high-value customers.

### 5. Payment Mode Analysis
- Compares usage of:
  - EMI
  - COD
  - Credit Card
  - UPI
  - Debit Card
- Provides insights into customer payment preferences.

### 6. Profit by Sub-Category
- Displays profitability of different products such as:
  - Phones
  - Bookcases
  - Stole
  - Printers
  - Shirts

## 💡 Business Insights

- Clothing contributes the highest quantity sold.
- Phones generate the maximum profit.
- February is the most profitable month.
- EMI and COD are the most preferred payment methods.
- Certain customers contribute significantly more revenue than others.
